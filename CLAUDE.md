# Happy Hour Discovery & Reservation Platform

A context-aware discovery platform that helps users go from search → compare → filter → reserve happy hour spots with minimal friction.

## Architecture

- **Frontend:** Next.js (`/web`)
- **Backend:** Node.js/TypeScript API (`/src`)
- **Database:** PostgreSQL
- **Infra:** AWS CDK (`/infra`), Railway
